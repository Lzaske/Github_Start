# GitHub Stars Hub

English entry page for GitHub Stars Hub.

Turn GitHub Stars into a static, searchable hub with sync, categorization, optional Simplified Chinese translation, and GitHub Pages deployment.

## What this project does

- fetches starred repositories from GitHub
- classifies repos with rule-based categories
- supports manual overrides in [`data/overrides.json`](data/overrides.json)
- optionally translates descriptions into Simplified Chinese
- builds and deploys a static site through GitHub Actions
